# 📝 Simple Text Editor (Java OOP)

A console-based text editor application built using Core Java and Object-Oriented Programming (OOP) concepts.  
This project demonstrates how text can be dynamically manipulated using Java’s StringBuffer class.

---

## 🚀 Features

- Add text  
- Reverse text  
- Delete a portion of text  
- Replace part of the text  
- Insert text at a specific position  
- Clear all text  
- Display current text  
- Show text length  
- Auto demo mode to test all features  

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)  
- Classes and Objects  
- Encapsulation  
- String manipulation using StringBuffer  
- User input handling using Scanner  

---

## 🏗️ Project Structure

- Document Class  
  Handles all text operations such as add, delete, replace, insert, reverse, and clear.

- TextEditor Class  
  Contains the main method and provides a menu-driven interface for user interaction.

---

## ▶️ How to Run

1. Compile the program:
   javac TextEditor.java

2. Run the program:
   java TextEditor

---

## 📋 Menu Options

1. Add Text  
2. Reverse Text  
3. Delete a Portion  
4. Replace a Portion  
5. Insert at Position  
6. Clear All Text  
7. Display Current Text  
8. Show Text Length  
9. Run Demo  
0. Exit

## 🧪 Demo Mode

The program includes an automatic demo mode that runs all features step-by-step to showcase functionality without manual input.
